# A Tour of Go の写経


## Go言語のインストール

下記のリンクから、**Featured downloads**の項目から、OSに合うインストールソフトをダウンロードする。

[https://go.dev/dl/](https://go.dev/dl/ "Go言語インストール")


### 環境構築(Windows)

#### A Tour of Go のインストール

* Go言語をインストールし、下記を実行。

```
go install golang.org/x/website/tour@latest
```

#### A Tour of Go の起動

```
tour.exe
```

#### プログラムの実行方法

```
go run example.go
```
